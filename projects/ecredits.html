E-credits

About

E-credits is an app developed for E-credits India, the first credit card due settlement company in India.This app allows user to settle their monthly credit card bills on time and also gets rewards for every settlement and referrals completed.

My role

The project was a group project, and I was the in charge of the app development from application UI Design to the frontend development using Flutter. I also assisted with the backend development process.

  • User Interface and Visual Style
  • Design System
  • Interactive Prototype
  • Frontend App Development
Tools and Technologies Used
  • Figma for UI design and design system
  • Flutter for app frontend development
  • REST api for network calls
  • PHP for backend development
  • MySQL for database.
Challenge

Clients previously owned a site to do the functionality, they needed an iOS and Android app to reach wider user base and better user experience. They also wanted to add additional functionalities like push notifications and referral system, which were not there on the website.

Solution

Our team understood all the client requirements carefully and came up with an application that worked exactly the way they wanted.
