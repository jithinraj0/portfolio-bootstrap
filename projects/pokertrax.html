PokerTrax.

About

The Poker Trax application will allow users to enter the location,duration, buy-in(money risked), and cash-out (money gained/lost) of the poker games they attend. The application will then use this information to show the users their hourly win/loss rate, overall profits, and understand what environments and scenarios they not only thrive in but also struggle in. The application is still in development phase.

My role

The project is a group project, and in the 1st sprint I was the in charge of the app design development.

  • User Interface and Visual Style
  • Design System
  • Interactive Prototype
Tools and Technologies Used
  • Figma for UI design and design system
  • Kotlin for android app development
  • RoomDB for database
Challenge

Our requirement was to come up with a simple but sophisticated app that serves its targeted users.

Solution

Our team understood all the requirements carefully and came up with an application that will help the users playing poker. We’ve developed an application with sophisticated features, yet elegant UI. The application is user friendly and facilitates guidance for amateur as well as professional poker players.
